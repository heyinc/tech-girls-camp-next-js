# 5. 商品詳細ページのレイアウトを改善しよう

商品詳細ページのレイアウトが寂しいですね。TailwindCSSを使ってレイアウトを改善してみましょう。


## やること

`src/app/items/[id]/page.tsx` を変更して、商品詳細ページのレイアウトを改善しましょう。

```diff
   return (
-    <main>
-      <Link href="/">← 商品一覧に戻る</Link>
-      <img src={item.image} alt={item.name} width={300} />
-      <h1>{item.name}</h1>
-      <p>{item.description}</p>
-      <p>{item.price.toLocaleString()}円</p>
-      <p>カテゴリー: {item.category}</p>
+    <main className="container mx-auto px-4 py-8">
+      <Link
+        href="/"
+        className="inline-block mb-8 text-blue-600 hover:text-blue-800"
+      >
+        ← 商品一覧に戻る
+      </Link>
+      <div className="grid grid-cols-1 md:grid-cols-2 gap-8">
+        <div>
+          <img
+            src={item.image}
+            alt={item.name}
+            className="w-full rounded-lg shadow-lg"
+          />
+        </div>
+        <div>
+          <h1 className="text-3xl font-bold mb-4">{item.name}</h1>
+          <p className="text-gray-600 mb-6">{item.description}</p>
+          <div className="bg-gray-100 p-6 rounded-lg">
+            <p className="text-3xl font-bold text-blue-600 mb-4">
+              {item.price.toLocaleString()}円
+            </p>
+            <p className="text-gray-600">カテゴリー: {item.category}</p>
+          </div>
+        </div>
+      </div>
     </main>
   );
 }
```

## 解説


###  `<main className="container mx-auto px-4 py-8">`

container: コンテンツの最大幅を設定し、レスポンシブなコンテナを作成します

[ドキュメント](https://tailwindcss.com/docs/container)

mx-auto: マージン

水平方向の余白を自動調整してコンテンツを中央揃えにします

[ドキュメント](https://tailwindcss.com/docs/margin)


px-4, py-8: パディング

左右に1rem, 上下に2remのパディングを設定します

[ドキュメント](https://tailwindcss.com/docs/padding)

### `<Link href="/" className="inline-block mb-8 text-blue-600 hover:text-blue-800">`

inline-block: インラインブロック要素として表示します

[ディスプレイのドキュメント](https://tailwindcss.com/docs/display)


mb-8: 下部に2remのマージンを設定します

[マージンのドキュメント](https://tailwindcss.com/docs/margin)


text-blue-600: テキストを青色(600)に設定します

[テキストカラーのドキュメント](https://tailwindcss.com/docs/text-color)


hover:text-blue-800: ホバー時にテキストをより濃い青色(800)に変更します

[ホバーのドキュメント](https://tailwindcss.com/docs/hover-focus-and-other-states)

## `<div className="grid grid-cols-1 md:grid-cols-2 gap-8">`

grid: グリッドレイアウトを使用します

[グリッドのドキュメント](https://tailwindcss.com/docs/grid-template-columns)


grid-cols-1: モバイル表示では1列にします

[グリッド列のドキュメント](https://tailwindcss.com/docs/grid-template-columns)


md:grid-cols-2: 中サイズ(768px以上)の画面では2列に分割します

[レスポンシブデザインのドキュメント](https://tailwindcss.com/docs/responsive-design)


gap-8: グリッドアイテム間に2remの間隔を設定します

[ギャップのドキュメント](https://tailwindcss.com/docs/gap)


## `<img src={item.image} alt={item.name} className="w-full rounded-lg shadow-lg" />`

w-full: 画像を親要素の幅いっぱいに広げます

[幅のドキュメント](https://tailwindcss.com/docs/width)


rounded-lg: 角を丸くします(大)

[角丸のドキュメント](https://tailwindcss.com/docs/border-radius)


shadow-lg: 大きめの影を追加してリッチな見た目にします

[シャドウのドキュメント](https://tailwindcss.com/docs/box-shadow)


## `<h1 className="text-3xl font-bold mb-4">{item.name}</h1>`

text-3xl: テキストサイズを大きく(1.875rem)設定します

[フォントサイズのドキュメント](https://tailwindcss.com/docs/font-size)


font-bold: フォントを太字にします

[フォントウェイトのドキュメント](https://tailwindcss.com/docs/font-weight)


mb-4, mb-6: 下部にマージン(1rem, 1.5rem)を設定して要素間のスペースを確保します

[マージンのドキュメント](https://tailwindcss.com/docs/margin)

## `<p className="text-gray-600 mb-6">{item.description}</p>`

text-gray-600: テキストを灰色(600)に設定して説明文を少し目立たなくします

[テキストカラーのドキュメント](https://tailwindcss.com/docs/text-color)


# 登場したHTMLの基本的なスタイリング概念の説明

## インラインブロックとは
インラインブロックは、要素の表示方法のひとつです。

インライン要素：<span>や<a>のように、テキストの流れの中に配置され、必要な幅だけを占めます。
ブロック要素：<div>や<p>のように、前後に改行が入り、幅いっぱいに広がります。
インラインブロック：インラインとブロックの特性を組み合わせたもので、テキストの流れの中に配置されながらも、幅や高さを設定できます。例えば、テキストの中にボタンを配置したいときに便利です。

## remとは
remは相対的な長さの単位です。

px：画面上のピクセル数を指定する絶対的な単位です。
rem：「root em」の略で、ルート要素（通常はhtml要素）のフォントサイズを基準にした相対的な単位です。

例：ルートのフォントサイズが16pxの場合、1rem = 16px、2rem = 32pxとなります。
画面サイズに応じて全体的に拡大縮小できるため、レスポンシブデザインに適しています。



## マージンとは
マージンは要素の外側の余白です。

要素と他の要素との間の「隙間」を作ります。
`margin-top`、`margin-right`、`margin-bottom`、`margin-left`で各方向のマージンを個別に設定できます。
例：`margin-bottom: 8px;`は要素の下に8pxの隙間を作ります。
Tailwindの`mb-8`は「下側に2rem（通常32px）のマージンを設定する」という意味です。

## パディングとは
パディングは要素の内側の余白です。

要素の境界線（ボーダー）と内容（コンテンツ）との間の「詰め物」のようなものです。
`padding-top`、`padding-right`、`padding-bottom`、`padding-left`で各方向のパディングを個別に設定できます。
例：`padding: 10px;`は要素の内側に全方向10pxの余白を作ります。
Tailwindの`p-6`は「全方向に1.5rem（通常24px）のパディングを設定する」という意味です。

簡単な例で理解する
箱（div要素）をイメージしてください：

コンテンツ：箱の中身（テキストや画像）
パディング：中身と箱の壁との間の空間
ボーダー：箱自体の壁（枠線）
マージン：箱と他の箱との間の距離

次のHTMLとCSSは、これらの概念を示しています：

```html
<div class="box">こんにちは</div>
```

```css
.box {
  display: inline-block; /* インラインブロック表示 */
  padding: 1rem; /* 内側に1remの余白 */
  margin: 2rem; /* 外側に2remの余白 */
  border: 1px solid black; /* 境界線 */
}
```

この例では、「こんにちは」というテキストの周りに内側の余白（パディング）があり、その箱の周りにさらに外側の余白（マージン）があります。箱はインラインブロックなので、必要な幅だけを占めつつ、パディングやマージンの設定が可能です。
